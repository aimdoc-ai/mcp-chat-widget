MCP Chat Widget

Web Component Implementation Example

How to Use

Include the script and add the custom element to your HTML:

<!-- Include the web component script -->
<script src="/dist/mcp-chat-widget.js"></script>

<!-- Use the web component in your HTML -->
<mcp-chat-widget 
  widget-id="123" 
  name="My Chat Widget"
  description="A customizable chat widget"
  position="bottom-right"
  size="md">
</mcp-chat-widget>

Available Attributes

Attribute Description Default
widget-id ID of the widget to load from the server Optional
name Name of the chat widget "Chat Assistant"
description Short description for the chat widget ""
system-prompt Custom system prompt for the AI Optional
default-provider Default AI provider Optional
position Position of the widget - "bottom-right" or "bottom-left" "bottom-right"
size Size of the widget - "sm", "md", "lg", "xl", or "full" "md"
mcp-servers JSON string of MCP servers configuration Optional
